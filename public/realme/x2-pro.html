realme X2 Pro

Unlocking and custom ROM installation

What's wrong with realme UI?

Everything. From battery bugs to memory leaks to stuttering in games to restricting features to their stock launcher, realme's software is insanely unoptimised.

Furthermore, their Privacy Policy and User Agreement states that they can (and do!) record everything you do on your device and send it to Chinese servers, and that you are not allowed to view or store pornography on your device.

How can we fix it?

The easiest way to fix this for your device is taking back control and installing a custom ROM. Some manufacturers, like realme, make this a pain to do, so you'll need some patience and the ability to read instructions.

Select your Android version

Some instructions vary based on the version of Android that you are running on your device. Selecting the version your device is running allows us to hide steps which don't apply to your version and add steps that do, so it is critical that you choose the right option.

Flashing OrangeFox recovery

You're running

Not right? Change your selected version

  1. Unplug your phone, then turn it off fully
  2. Hold Volume Down and the Power button to boot the phone into Fastboot mode
  3. Plug the phone back into your computer
  4. Download OrangeFox Recovery
    • OrangeFox is a skinned version of TWRP recovery with added features and a simpler UI. If you have issues, you're more likely to find a solution by using TWRP in your Google search than OrangeFox. Note that options are in different locations in OrangeFox and TWRP so solutions might not work without some looking around.
  5. Download vbmeta.img
  6. Move both of these files to the same directory as ADB and Fastboot
  7. In your command prompt, run these commands to install OrangeFox and flash your vbmeta. It should say successful after each command. 
    fastboot flash recovery OrangeFox-R10.1_1.img
fastboot flash vbmeta vbmeta.img
  8. On your phone, press Volume Down until you see RECOVERY MODE at the top, then press Power

Downgrade to ColorOS firmware

Due to how most Custom ROMs are made for the X2 Pro, you need to downgrade to the ColorOS firmware and vendor files before being able to run them.

Warning.

After doing this, you will not be able to boot into realme UI because you'll have some files from Android 9 and some from Android 10. This is normal.

  1. Download the Pie firmware and vendor zip for your device
    • Global devices
    • Chinese devices
  2. Move this zip file to the same folder as your ADB and Fastboot
  3. Push this zip file to your phone: 
    adb push -p "<FILE NAME>" "/sdcard"
    • Replace <FILE NAME> with the name of the zip file that you downloaded
  4. On your phone, after the push is complete, tap Files (even if already selected), then scroll and tap the zip you just pushed. Swipe the bar to flash it.

Flashing AOSiP

This guide is specific to the AOSiP Custom ROM. If you know what you're doing, you can adapt this to almost any other Custom ROM you want. At the moment, AOSiP seems to be the most stable and have the most features.

  1. Download the latest build of AOSiP
    • This can be found at the link above, under the Downloads header. Top one is most recent.
  2. If you use VoLTE, also download the VoLTE fix
  3. Download OpenGApps (Google apps)
  4. Move both of these files to the same folder as ADB and Fastboot
  5. In your command prompt, enter this command: adb push -p "<AOSiP FILE NAME>" "/sdcard"
    • Replace <AOSiP FILE NAME> with the name of your AOSiP zip file that you downloaded
  6. Wait for this command to complete. This may take several minutes. The progress will be shown in your command prompt window.
  7. On your phone, tap Files (even if already selected) at the bottom of OrangeFox
  8. Scroll down and find your AOSiP zip file and tap it, then swipe the bar to flash the ROM
  9. On your phone, tap Menu > Manage Partitions > Data > Format Data > Type yes > Press (tick)

    If you see a red error, you must go to Menu > Reboot > Recovery, and try this step again.

  10. If you use VoLTE, push the file, then use the same method as above to flash it.
    • Use this command to push the VoLTE fix file, using the correct file name in place of the placeholder 
      adb -p push "<VoLTE FIX FILE NAME>" "/sdcard"
  11. Next, push the OpenGApps zip file to your device: adb push -p "<OpenGApps ZIP NAME>" "/sdcard"
    • Replace <OpenGApps ZIP NAME> with the name of your OpenGApps zip file that you downloaded

    If you see a red error, you must go to Menu > Reboot > Recovery, and try this step again.

  12. On your phone, after the push is complete, tap Files (even if already selected), then scroll and find your OpenGApps zip. Tap it, and swipe the bar to flash it.
  13. After the flash is complete, tap Menu, then Reboot, then System
  14. If you want Magisk for root access, download Magisk 20.3, then push and flash it like the other files

    You don't need Magisk to pass SafetyNet on AOSiP, but you might on other ROMs. Note that Magisk is helpful for other things, like Substratum themes and Fluid Navigation Gestures.

    Some people have reported issues with flashing Magisk before booting once, so it's recommended that your boot at least once before flashing Magisk.

    If you see a red error, you must go to Menu > Reboot > Recovery, and try this step again.

Reverting to ColorOS or realme UI

You're running

Not right? Change your selected version

  1. Unplug your phone, then turn it off fully
  2. Hold Volume Down and the Power button to boot the phone into Fastboot mode
  3. Plug the phone back into your computer
  4. On your phone, press Volume Down until you see RECOVERY MODE at the top, then press Power
  5. Download the OZIP file for the Android variant you want to use. You can find all realme OZIPs on Realme Updater (links below).
    • Axx is ColorOS 6. Cxx is realme UI 1.0.
    • Global devices
    • Chinese devices

    You should be able to flash the Global OZIP on a Chinese device and vice versa using this method.

    If something goes wrong, just reflash OrangeFox and flash the OZIP designed for your device.

  6. In your command prompt, enter this command to "push" (transfer) the OZIP update file to your phone: adb push -p "<OZIP FILE NAME>" "/sdcard"
    • Replace <OZIP FILE NAME> with the name of your OZIP file that you downloaded
  7. In OrangeFox, after the push is complete, tap Files at the bottom (even if already selected)
  8. Scroll and tap on your OZIP file, then swipe the bar to flash it
  9. Tap Menu > Manage Partitions > Data > Format Data > Type yes > Press (tick)
  10. Tap Menu > Reboot > System
  11. If you want to re-lock your bootloader, download the in-depth test APK (at the start of this guide), and press Apply to exit in-depth test. Follow the instructions in the app.
    • If you have an unlocked bootloader, you'll only have L3 Widevine DRM (no HD Netflix), and you may not pass SafetyNet (no Netflix or Google Pay at all)

    This could brick your device if you still have traces of Magisk or OrangeFox on your device.
