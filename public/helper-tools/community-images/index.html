      Community image URLs?

      Enabling community image URLs is strongly recommended.

      This feature uses URLs which are hosted on the community instead of this website. This allows members to access the images you share when they have no data. It also means your image links will still work if this website stops running, or changes domain.

      However, it also means that your images will not retrospectively update and, if a new version of an image is released, you will need to manually copy the image code again.
